# Engineering Docs

These docs are reusable project defaults. Application-specific decisions belong in `project/PROJECT.md`, `project/DECISIONS.md` and module docs.

Read only the docs relevant to the active work order.

For a stakeholder-friendly walkthrough that can be exported to PDF or DOCX, use
`docs/walkthrough/README.md`. It is user-facing documentation and is not
mandatory operational context for agents.
