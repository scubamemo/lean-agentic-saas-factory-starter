# Workflow

Start from `START-HERE.md`, `AGENTS.md` and `project/work-orders/active-work-order.md`. Do not scan the full repo.

## Bugfix

1. Reproduce or describe the bug.
2. Identify the smallest affected module.
3. Read module contracts and handoff.
4. Patch the smallest allowed implementation set.
5. Add or update regression test notes in `test-matrix.md`.
6. Update `handoff.md`.
