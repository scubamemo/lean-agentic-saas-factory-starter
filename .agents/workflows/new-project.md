# Workflow

Start from `START-HERE.md`, `AGENTS.md` and `project/work-orders/active-work-order.md`. Do not scan the full repo.

Apply `.agents/rules/hook-policy.md`: pre-read before project discovery,
pre-write before project artifact edits, and pre-handoff before creating or
routing the first work order.

## New project

1. Fill `project/PROJECT.md`.
2. Fill `project/UI.md`.
3. Define modules in `project/MODULES.md`.
4. Select profile in `project/project.config.json`.
5. Run `pnpm check` or manually validate required docs.
6. Create first active work order.
