# Project Instantiation

1. Copy this repository.
2. Remove any local/release artifacts ignored by `.templateignore`.
3. Fill `project/PROJECT.md`, `project/UI.md`, `project/MODULES.md`.
4. Set `project/project.config.json`.
5. Run `pnpm check`.
6. Create the first real module from `project/modules/_template`.
7. Fill `project/work-orders/active-work-order.md`.
8. Start Antigravity with `START-HERE.md` and `AGENTS.md`.
